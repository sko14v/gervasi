## 2. Comisiones — estructura escalonada

| Evento | Comisión | Detalle |
|--------|---------|---------|
| **Agenda generada** | 10 EUR | Prospecto acepta la reunión con el closer |
| **Show** (se presenta) | 50 EUR total | La agenda de 10 EUR se convierte a 50 EUR |
| **Cierre** (firma) | 100 EUR total | Se completa a 100 EUR |

### Cómo funciona

```
Prospecto acepta reunión
  → Se registra como "agenda"
  → Xisco cobra: 10 EUR
  │
  └→ Si el prospecto SE PRESENTA
       → Se marca como "show"
       → Xisco cobra: +40 EUR extra (total 50 EUR)
       │
       └→ Si se CIERRA (firma con Growing)
            → Se marca como "cierre"
            → Xisco cobra: +50 EUR extra (total 100 EUR)
```

### Citas canceladas y reagendadas

| Situación | Acción |
|-----------|--------|
| Se cancela antes del show | Se puede reagendar sin penalización |
| Se reagenda y finalmente se presenta | Se cobra el show completo (50 EUR) |
| Se reagenda y se cierra | Se cobra el cierre completo (100 EUR) |

> **Nota:** Xisco marca manualmente en el CRM de Growing:
> agenda, show/no-show, cierre, cancelación/reagendación.

## 5. Acciones si se está por debajo del objetivo

| Condición | Acción |
|-----------|--------|
| Llamadas < 80% del objetivo semanal | Revisar FIPAs, identificar qué fase del script falla |
| Show rate < 60% | Trabajar el paso de cierre (2 opciones de horario + datos) |
| Agendas < 80% del objetivo | Revisar filtro de dinero y tiempo — probablemente se están saltando |
| Score promedio < 70 | Feedback Coach genera plan de mejora individual |
| Score promedio < 50 | Escalar a coaching 1:1 |
